பரந்த வானில் <br/>
விரிந்த சிறகில் <br/>
பிறந்த கவியில் <br/>
வளர்ந்த உறவில் <br/>
புதைந்த நொடிகள் <br/>
புதையல் என்பேன்.

கலைந்த கனவில் <br/>
உடைந்த மனதில் <br/>
சிதைந்த உலகில் <br/>
தொலைந்த வழியில் <br/>
மறந்த நொடிகள் <br/>
மரணம் என்பேன்.

கோபங்கள் போதும் <br/>
கொஞ்சல்கள் வேண்டும் <br/>
வாதங்கள் போதும் <br/>
வாஞ்சனைகள் வேண்டும் <br/>
மோதல்கள் போதும் <br/>
மோகங்கள் வேண்டும் <br/>
சோகங்கள் போதும் <br/>
சொர்க்கங்கள் வேண்டும்

சிறகுகள் முளைக்கட்டும் <br/>
சிறுவானில் சேர்ந்திருப்போம் <br/>
மறுவாழ்வு உள்ளதெனில் <br/>
மீண்டுமதில் மகிழ்ந்திருப்போம்.

வா!
