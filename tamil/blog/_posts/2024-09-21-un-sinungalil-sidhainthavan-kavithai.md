title: உன் சிணுங்களில் சிதைந்தவன்
date: 2024-09-21
category: tamil-blog
---

சந்தனத் தமிழை <br/>
சிந்திடும் இதழோ <br/>
என்னுடன் பேசிடும் <br/>
உன்னிரு விழியோ

மெல்லிசை பாடிடும் <br/>
புன்னகை மலரோ <br/>
கொஞ்சிடும் கூந்தலில் <br/>
பூத்திடும் நிலவோ

தன்னிலை மறந்து <br/>
தாளங்கள் தீட்டிடும் <br/>
கன்னியின் விரலோ <br/>
சிந்திய தூறலோ

கட்டிய சேலையில் <br/>
கொட்டிய பூக்களும் <br/>
சிட்டு நீ பாடவே <br/>
மெட்டுக்கள் தேடுதோ

ஒற்றையில் ஆடிடும் <br/>
பொட்டிடும் அழகோ <br/>
கற்றனைத் தூறிய <br/>
காவியக் கவியோ

சித்தன்ன கோயிலின் <br/>
சித்திர சிலையோ <br/>
சந்தங்கள் சேர்ந்திடும் <br/>
முத்தமிழ் மழையோ

&#8722; உன் சிணுங்களில் சிதைந்தவன்
